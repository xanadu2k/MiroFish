# 关键词与变量设计（用于 MiroFish 配置）

## A. 核心关键词（可直接粘贴）

船舶检验, AI船检, 无人机巡检, 无人机器狗巡检, 无损检测, 缺陷识别, 智能诊断, 船级社认证, 合规检验, 验船数字化, 船队降本增效, 海事安全, 高危工况替代, 多船型适配, 运油船, 集装箱船, 散货船, 国资航运客户, 规模化落地, 检验报告自动生成

## B. 平台-行业-市场 三维关键词

- 平台维度：多模态检测数据, 边缘采集, AI分析引擎, 报告自动化, 数据追溯, 作业调度
- 行业维度：船级社, 设备认证, 检验规范, 合规审查, 海事监管, 风险分级
- 市场维度：大型船队, 200条船, 成本压力, 安全提升, 交付周期, 标杆客户

## C. Agent 角色建议

- 创业者（Founder）：推动战略、资源整合、融资与业务拓展
- 技术负责人（CTO）：设备-算法-平台协同与产品化
- 船级社评审专家：关注合规性、认证标准、可审计性
- 船东运营总监：关注成本、船期、服务可用性
- 现场检验负责人：关注可操作性、作业安全、效率
- 风控/法务：关注责任边界、数据合规、合同条款
- 行业竞争对手：模拟替代方案与价格/服务竞争
- 投资人：关注可复制性、单位经济性与增长速度

## D. 不确定性变量（用于推演分支）

- 认证进度：快 / 中 / 慢
- 客户采用率：高 / 中 / 低
- 设备稳定性：稳定 / 波动 / 关键故障
- AI准确率：高 / 中 / 低
- 政策环境：支持 / 中性 / 收紧
- 市场竞争：温和 / 加剧
- 资金状况：充裕 / 一般 / 紧张

## E. 成功判定指标（建议纳入模拟目标）

- 单船检验成本下降比例
- 单船检验周期缩短比例
- 高风险人工工时替代比例
- 设备作业成功率与复检率
- 船级社认证节点达成率
- 客户续约率/扩单率
- 单月可交付船检数量
- 单位项目毛利率

## F. 失败预警信号（建议纳入监控）

- 认证反复延迟且无替代路径
- 关键客户试点迟迟不扩面
- 设备故障导致作业中断频发
- AI结论与人工复核偏差过大
- 单船交付成本持续高于传统模式
